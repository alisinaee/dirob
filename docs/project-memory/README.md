# Project Memory (Torob)

This folder is the persistent memory source for new chats in this repository.

## Files
- `chat-summaries.md`: short per-session summaries.
- `bug-log.md`: known bugs, status, repro, and fix notes.
- `work-log.md`: what changed, where, and why.
- `decisions.md`: architectural and product decisions.

## Usage Contract
- Read these files at the start of new work.
- Append concise entries after meaningful changes.
- Keep entries factual, implementation-focused, and timestamped.
